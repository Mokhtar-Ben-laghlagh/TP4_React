# 📰 Mon Blog React

Application React de gestion d'articles avec navigation, CRUD complet et Bootstrap.

---

## 🚀 Installation

```bash
npm install
npm install bootstrap
npm start
```

---

## 📁 Structure

```
src/
├── pages/
│   ├── Accueil.js
│   ├── Article.js
│   ├── AddArticle.js
│   └── EditArticle.js
├── components/
│   ├── ArticleList.js
│   └── ArticleDetail.js
├── App.js
└── index.js
```

---

## 🔗 Routes

| URL                 | Page                |
|---------------------|---------------------|
| `/`                 | Liste des articles  |
| `/article/:id`      | Détail d'un article |
| `/article/:id/edit` | Éditer un article   |
| `/ajouter`          | Nouvel article      |

---

## ✅ Fonctionnalités

- Afficher la liste des articles
- Consulter le détail d'un article
- Ajouter un article
- Éditer un article
- Supprimer un article
- Validation des champs vides

---

## 🛠️ Technologies

- React 18
- React Router DOM
- Bootstrap 5
